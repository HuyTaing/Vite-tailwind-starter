# vite & Tailwind Starter (Novembre 2022)

Starter Tailwindcss avec sass, postcss & autoprefixer

### Version
vite : 3.2.3
tailwindcss : 3.2.2

## Usage

### Install dependencies

```
npm Install
```

### Run the development server

```
npm run dev
```

### Build for production

```
npm run build
```

### Build for preview

```
npm run preview
```
